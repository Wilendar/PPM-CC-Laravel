**Solution:**

- ‚úÖ Upsert pattern: `updateOrCreate()` zamiast `create()`
- ‚úÖ Error handling: catch unique violation exceptions

### **Problem 4: JSON Column Query Performance**

**Issue:** `predefined_values` JSON queries mogƒÖ byƒá slow

**Solution:**
- ‚úÖ Laravel 12.x JSON casting: automatic serialization/deserialization
- ‚úÖ Avoid querying JSON columns directly: use indexed columns instead
- ‚úÖ Denormalization: cache frequently queried JSON values

**Avoid:**

**Use Instead:**

### **Problem 5: Cascade Delete Performance on Large Datasets**

**Issue:** `onDelete('cascade')` mo≈ºe timeout dla 10K+ related records

**Solution:**
- ‚úÖ Queue-based deletion dla large datasets
- ‚úÖ Batch deletion: chunk related records
- ‚úÖ Soft deletes: avoid hard deletes
- ‚úÖ Background job: async cleanup

**Code Pattern:**
